# ロードマップ

## マイルストーン概要

```
v0.1 MVP ──── v0.5 整理・振り返り強化 ──── v1.0 正式リリース ──── v1.x 共有拡張
```

---

## v0.1 MVP（個人利用・検証フェーズ）

**目標**: 家族のリーダーが毎日使える最低限の記録機能を実現する

- [ ] テキスト記録の作成・編集・削除
- [ ] 写真の添付（最大5枚/エントリ）
- [ ] 日付による記録一覧表示
- [ ] ローカル認証（パスワード or 生体認証）
- [ ] データのローカル保存
- [ ] 家族ごとの記録対象ラベル管理

---

## v0.5 整理・振り返り強化（継続利用フェーズ）

**目標**: 記録を整理しやすくし、気遣いに活かせる状態を作る

- [ ] タグ付け・フィルタ機能
- [ ] 家族別ビュー
- [ ] プッシュ通知（振り返り通知）
- [ ] 検索機能（日付・タグ・家族）
- [ ] 月次の振り返りビュー

---

## v1.0 正式リリース

**目標**: 安心して長期利用できる品質と移行性を整える

- [ ] データエクスポート（JSON / PDF）
- [ ] バックアップ・リストア機能
- [ ] プライバシーポリシーの公開
- [ ] パフォーマンス最適化
- [ ] App Store / Google Play 審査対応
- [ ] クラウドバックアップの検討

---

## v1.x 共有拡張（ポストリリース）

- 家族グループの作成・招待
- 共有閲覧機能
- 共有リアクション機能
- クラウド同期
- 月次・年次レポート生成
- 音声メモ添付
- 検索機能の強化（全文検索）
- 子ども向けUI（アクセシビリティ）
- Web版対応

---

## 優先度の考え方

1. **使い続けられること** > 機能の多さ
2. **プライバシー・セキュリティ** は常に最優先
3. **記録のしやすさと見返しやすさ** を共有機能より優先する
4. **データ移行・エクスポート** はv1.0前に必ず実装（ロックイン防止）
