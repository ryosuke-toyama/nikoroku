# 要求定義

## ユーザーストーリー

### 記録
- 家族のリーダーとして、自分や家族について気になったことを短い文章で素早くメモしたい
- 家族のリーダーとして、家族の様子や出来事を写真付きで記録したい
- 家族のリーダーとして、家族ごと・テーマごとに記録を分類したい
- 家族のリーダーとして、過去の記録を日付・人・タグで検索して見返したい

### 把握・振り返り
- 家族のリーダーとして、家族ごとの最近の記録をひと目で把握したい
- 家族のリーダーとして、約束や気がかりを忘れないように見返したい
- 家族のリーダーとして、「今日の1年前」など過去の記録を振り返りたい

### 管理・セキュリティ
- 家族のリーダーとして、記録へのアクセスをパスワード・生体認証で保護したい
- 家族のリーダーとして、全データをJSONまたはPDF形式でエクスポートしたい
- 将来的な管理者として、必要になった段階で家族グループを作成し、共有範囲を制御したい

## 機能要件

| 優先度 | 機能 | 説明 |
|---|---|---|
| Must | テキスト記録 | 文章・箇条書きで日記を書く |
| Must | 写真添付 | 1エントリに複数枚の写真を添付 |
| Must | 一覧表示 | 自分が残した記録を時系列で閲覧 |
| Must | ローカル保護 | パスワードまたは生体認証で記録を保護 |
| Should | タグ付け | 自由タグでの分類・フィルタ |
| Should | 家族別管理 | 家族ごとに記録や関心事項を整理 |
| Should | 振り返り通知 | 過去の同日記録や見返し候補の通知 |
| Could | 家族グループ | 招待制のプライベートグループ |
| Could | 共有リアクション | 他メンバーからの閲覧・リアクション |
| Could | 月次/年次レポート | PDF形式の振り返りレポート生成 |
| Could | 音声メモ | 音声録音の添付 |

## 非機能要件

| 項目 | 要件 |
|---|---|
| パフォーマンス | 記録一覧の表示 2秒以内 |
| 可用性 | 月次稼働率 99.5% 以上 |
| セキュリティ | 通信の全暗号化(TLS 1.3+)、保存データの暗号化 |
| プライバシー | 家族やグループに関する記録の漏洩ゼロ |
| データ保全 | 日次バックアップ、最低30日間の保持 |
| エクスポート | JSON / PDF形式でのフルエクスポート対応 |
