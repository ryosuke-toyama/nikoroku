# ユーザーフロー・画面遷移

## 主要フロー

### 新規ユーザーフロー

```
アプリ起動
  └─ 初回起動
        ├─ オンボーディング
        ├─ アプリロック設定（パスコード or 生体認証）
        ├─ 記録対象の初期登録（自分 / 家族）
        └─ ホームへ

再起動
  └─ ロック解除 → ホームへ
```

### 記録作成フロー

```
ホーム（記録一覧）
  └─ [+ボタン] → エントリ作成画面
        ├─ 日付選択（デフォルト: 今日）
    ├─ 記録対象選択
        ├─ テキスト入力
        ├─ 写真追加（任意）
        ├─ タグ付け（任意）
    └─ [保存] → 記録一覧に反映
```

### 振り返りフロー

```
ホーム
  ├─ [記録対象] → 対象ごとの記録一覧
  └─ [カレンダー] → 月次カレンダービュー
        └─ 日付タップ → その日のエントリ一覧
              └─ エントリタップ → 詳細表示
```
